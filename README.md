### 2.怎么安装
#### 2.1 修改Makefile中的一些值
* <b>THEOS_DEVICE_IP: </b> You JailBreak iOS device's ip, staying in the same wifi network with your mac
* <b>TARGET: </b> iOS SDK version you use

#### 2.2 安装
```
make package install
```
